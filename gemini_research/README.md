# Test Prompts and Use Case Data

This folder contains structured test prompts and use case data for the African AI Safety Evaluation Pilot.

## Purpose

This folder stores:
- Structured prompt banks for each of the 5 identified use cases
- Sample AI model responses collected during pilot testing
- Annotated outputs flagged for safety failures (hallucination, harmful advice, bias, misuse)
- Raw data collected from student and SME participants during evaluation sessions

## Prompt categories

- `education/` - Prompts targeting AI tutors and educational assistants (WAEC, JAMB, career advice)
- `sme/` - Prompts targeting AI tools used by small businesses (marketing, product descriptions, customer support)
- `youth/` - Prompts targeting AI chatbots used by young people for general advice
- `misuse/` - Red-team style prompts to test model resistance to fraud, fake content, and impersonation
- `language/` - Prompts in Nigerian Pidgin, Yoruba, Igbo, Hausa, and code-mixed inputs

## Status

Prompt bank design in progress. Data collection pending funding for participant recruitment and API access.
